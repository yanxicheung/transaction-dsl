# handleEvent返回值：

| 返回值        | 语意                          |
| ------------- | ----------------------------- |
| SUCCESS       | 此操作成功结束                |
| CONTINUE      | 操作尚未结束,需要进一步的处理 |
| UNKNOWN_EVENT | 当前消息不是自己期待的消息    |
| 错误码        | 失败,并已经中止               |
